## 1. 브랜드 포지셔닝

### 우리는 무엇인가
매쉬업벤처스는 **창업자가 만든 VC**다.
이택경(Daum 공동창업자)이 설립하고, 창업 경험을 가진 파트너들이 운영하며,
50명 이상의 선배 창업자 네트워크가 포트폴리오사를 실질적으로 돕는다.

단순히 돈을 주는 투자자가 아니라,
**가장 이른 순간부터 가장 깊이 함께하는 파트너**다.

### 누구를 위한 VC인가
- 아이디어 단계 or 초기 MVP의 IT 창업자
- 사업계획서보다 실행력과 비전이 있는 팀
- 자본만큼 네트워크와 경험이 필요한 시점의 스타트업

### 무엇이 다른가
| 일반 VC | 매쉬업벤처스 |
|---------|------------|
| 심사역이 투자 결정 | 창업자 출신 파트너가 직접 판단 |
| 투자 후 quarterly 보고 | 50+ 멘토 네트워크, 오피스아워, 세미나 |
| 시리즈A 이후 선호 | Pre-seed ~ Seed, 가장 이른 단계 |
| 포트폴리오 = 리스트 | 포트폴리오 = 커뮤니티 |

## 5. 브랜드 보이스 & 톤

### 원칙
| 방향 | 설명 |
|------|------|
| **확신 있게** | 조심스럽지 않다. 우리가 왜 좋은지 말할 수 있다 |
| **창업자의 언어로** | VC 관용어 지양. 창업자가 쓰는 단어로 |
| **간결하게** | 긴 설명보다 짧고 강한 문장 |
| **따뜻하지만 무게감** | 친근하되 가볍지 않다 |

### 피해야 할 것
- "혁신적인", "다양한", "시너지" 류의 빈 형용사
- VC 투자 용어 남발 (LP, IRR, 멀티플 등 — 필요한 곳에만)
- 과도한 겸손 ("함께 성장하고 싶습니다" 식)
- 지나치게 영미권 흉내낸 영어 카피

### 한/영 카피 원칙
- 영어는 한국어 번역이 아닌 **독립 재작성**
- 한국어: 간결하고 선명한 현대 한국어
- 영어: 간결하고 자신감 있는 영어 (Sequoia 톤 참고)

## 7. 컬러 시스템 방향 (3가지 옵션)

> **베이스는 라이트로 확정.** 포인트 컬러 방향만 결정 필요.

---

### Option A — Warm Amber (추천)
```
Background:  #F7F5F0  (따뜻한 오프화이트)
Surface:     #FFFFFF
Text:        #111111
Accent:      #C8781A  (번트 앰버)
Gray:        #6B6B6B, #D4D0C8
```
**무드:** 따뜻함, 창업자 에너지, 예상 밖, 격식 없는 권위
**왜 Mashup에 맞나:** "초기" + "커뮤니티" 브랜드 키워드와 연결. 시드 단계의 황금 같은 가능성. VC 사이트에서 보기 드문 컬러 → 기억에 남음. Instrument Serif와 황금 콤보.

---

### Option B — Deep Indigo
```
Background:  #F8F8FF  (쿨한 화이트)
Surface:     #FFFFFF
Text:        #0F0F1A
Accent:      #3730A3  (딥 인디고)
Gray:        #6B7280, #E5E7EB
```
**무드:** 기술적 권위, 깊이감, 전문성
**왜 Mashup에 맞나:** a16z 퍼플과 유사한 계열. AI/테크 중심 포트폴리오에 어울림. 진지함과 포워드-씽킹 동시 표현.

---

### Option C — No Accent (Pure Editorial)
```
Background:  #FAFAF8  (뉴트럴 오프화이트)
Surface:     #FFFFFF
Text:        #111111
Accent:      없음 (타이포그래피 + 여백이 전부)
Gray:        #555555, #CCCCCC
```
**무드:** 극단적 미니멀, 에디토리얼 권위, 폰트가 모든 것
**왜 Mashup에 맞나:** Instrument Serif가 가장 빛남. 색이 없으면 카피와 사진이 살아남. 가장 "시니어"하고 자신감 있는 선택.
**리스크:** 차가워 보일 수 있음 — 창업자 친화적 무드 약화 가능성.

---

### 컬러 비교

| | Option A (Amber) | Option B (Indigo) | Option C (No Accent) |
|--|--|--|--|
| 따뜻함 | ★★★★★ | ★★★ | ★★ |
| 힙함 | ★★★★★ | ★★★★ | ★★★★ |
| 권위 | ★★★★ | ★★★★★ | ★★★★★ |
| 차별성 | ★★★★★ | ★★★ | ★★★★ |
| 창업자 친화 | ★★★★★ | ★★★★ | ★★★ |

> **추천: Option A (Warm Amber)** — 브랜드 키워드(EARLY, FOUNDER, COMMUNITY)와 가장 잘 맞고, VC 씬에서 가장 차별화됨. Instrument Serif + 앰버 = 기억에 남는 조합.

## 8. 타이포그래피 확정

| 역할 | 폰트 | 비고 |
|------|------|------|
| Display / Hero (EN) | **Instrument Serif** | Google Fonts, 무료 |
| Body / UI (EN) | **Plus Jakarta Sans** | Google Fonts, 무료 |
| 전체 (KO) | **Pretendard** | 무료, 최고의 한국어 폰트 |
| 모노스페이스 (레이블용) | **JetBrains Mono** or **IBM Plex Mono** | 선택적, BlueBrown식 코드 레이블에 활용 |

```

H2:          36px / Plus Jakarta Sans / SemiBold

Body L:      18px / Plus Jakarta Sans / Regular
Body M:      16px / Plus Jakarta Sans / Regular
Caption:     13px / Plus Jakarta Sans / Regular
Label/Tag:   11px / Plus Jakarta Sans / Medium + Letter-spacing 0.08em
```
