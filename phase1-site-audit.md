# Phase 1 — 현황 분석: mashupventures.co 사이트 감사

> 감사 일자: 2026-03-31
> 대상: https://www.mashupventures.co (홈 + 서브 5페이지)

---

## 1. 기술 스택

| 항목 | 현황 |
|------|------|
| 빌더 | **Webflow** (No-code CMS) |
| JS | jQuery 3.5.1 + Webflow 자체 번들 |
| 폰트 로딩 | WebFont Loader (Google Fonts — 동기 로딩, 성능 저하 위험) |
| 분석 | Google Tag Manager (GTM-PXVG67H7) |
| 이미지 CDN | Webflow CDN (website-files.com) |
| 채용 | RoundHR (외부 서비스 연동) |
| 폼 | forms.mashupventures.co (별도 서브도메인) |
| SEO | 네이버 사이트 확인 + canonical + OG/Twitter Card |

**기술 부채 포인트:**
- Webflow 종속성 — 커스텀 코드 한계, 퍼포먼스 최적화 어려움
- jQuery 로딩이 동기 방식 → 초기 렌더 블로킹 가능성
- WebFont 동기 로딩 → FOUT/FOIT 발생 가능

## 2. 현재 디자인 시스템

### 2-1. 컬러

- 블루 계열 컬러가 5가지 이상 혼재 — 브랜드 컬러 불명확
- 토큰화된 컬러 시스템 부재 → 일관성 없음
- 접근성 검토 미확인

| Section Title | 32px | 섹션 구분 |
| Sub Heading | 24px | |
| Body | 15–16px | |
| Caption / Small | 10–12px | |

**폰트:** Noto Sans KR (300~700) + Material Icons

**문제점:**

## 3. 페이지별 현황 분석

### 3-1. 홈 (/)

**섹션 구성:**
1. Hero — "선배 창업자가 미래의 창업자를 지원합니다"
2. 포트폴리오 그리드 (206개사 로고)
3. 투자 성과 지표 (206개사 / 11.3조원 / 2.4조원 / 12개사 IPO·M&A)
4. 투자 전략 (3가지 핵심 기준)
5. TIPS 프로그램 (800M~1.5B원, 86명 선발)
6. 가치성장 네트워크 (50+ 멘토)
7. 창업자 후기 (3개)
8. 최근 콘텐츠
9. CTA + 푸터

**강점:**
- 핵심 KPI 수치가 명확하게 노출됨
- 창업자 후기로 신뢰 구축
- 투자 신청 CTA가 항상 상단에 고정

**개선 필요:**
- Hero 비주얼이 밋밋 — 텍스트 중심, 임팩트 부족
- 포트폴리오 로고 그리드가 지나치게 나열식 — 스토리 없음
- 섹션 간 리듬/흐름이 단조로움
- 스크롤 인터랙션 거의 없음
- 영어 이용자 고려 없음 (한국어 전용)

---

### 3-2. 팀 (/team)

**구성:** 창업자/대표 → 파트너 4 → AI 파트너 → Venture Partners → 투자팀 → 커뮤니케이션 → 운영 → TIPS팀 → EIR → 어드바이저

**강점:**
- 팀 전체 구조 투명하게 공개
- 역할 구분이 명확

**개선 필요:**
- 개인 프로필 클릭 시 확장 정보 여부 불명확 (상세 bio 부재 가능성)
- 팀원 소개에 스토리/개성 없음 — 이름+직책 나열 수준
- 각 파트너의 투자 포커스/전문 분야 미노출

---

### 3-3. 포트폴리오 (/portfolio)

**구성:** 성과 지표 → 카테고리 필터(18개) → 포트폴리오 카드 → 페이지네이션

**강점:**
- 18개 카테고리 필터로 섹터별 탐색 가능
- IPO/M&A 배지로 엑싯 성과 직관적 표현

**개선 필요:**
- 포트폴리오 카드에 회사 소개 없음 — 로고+배지만 있어 맥락 부족
- 페이지네이션 방식 — 무한 스크롤 or 더 직관적인 탐색으로 개선 여지
- 주목할 만한 포트폴리오사를 강조하는 큐레이션 없음

---

### 3-4. 가치성장 네트워크 (/growth-network)

**구성:** Hero → 어드바이저 프로필(4명+50여 명) → 3가지 프로그램(오피스아워/세미나/전문서비스) → 후기 캐러셀

**강점:**
- 3가지 서비스 구조가 명확
- 실제 어드바이저 이름/사진으로 신뢰감 형성

**개선 필요:**
- 어드바이저 프로필 정보 부족 — 어떤 분야 전문가인지 불명확
- 창업자가 실제로 어떤 도움을 받을 수 있는지 구체성 부족
- 50+ 어드바이저 전체 리스트 미공개 — 신뢰성 기회 손실

---

### 3-5. 콘텐츠 (/contents)

**구성:** 추천 콘텐츠(3개) → 4개 카테고리(스타트업 가이드/투자관점/창업자 스토리/매쉬업 뉴스)

**강점:**
- 카테고리 구분으로 콘텐츠 탐색 용이
- 무료 PDF 배포 등 리드 마그넷 요소 있음

**개선 필요:**
- 콘텐츠 양이 충분한지 불명확 — 블로그 활성도 확인 필요
- 검색 기능 없음
- 태그/필터 기능 없음
- 콘텐츠별 예상 읽기 시간 미표시

---

### 3-6. 투자검토 신청 (/apply)

**구성:** 신청 안내 → 투자 기준 → FAQ → 신청 폼

**강점:**
- "사업계획서 불필요" 메시지로 진입 장벽 제거
- FAQ로 창업자 불안 요소 선제 해소
- 심사 기간 (1~2개월) 명시로 기대치 관리

**개선 필요:**
- 폼 UI/UX 미확인 — 실제 신청 경험 검토 필요
- 투자 단계/금액 정보가 너무 간략 (최대 5억원, Seed 집중)
- 심사 절차 시각화 없음 (단계별 프로세스 미노출)

## 5. 개선 방향 요약

```
현재: 정보 나열형, Webflow 기반, 컬러/타입 체계 부재
목표: 스토리텔링형, 체계적 디자인 시스템, 힙하고 세련된 VC 브랜드
```

**핵심 방향:**
1. **브랜드 재정의** — 블루 컬러 정리 or 완전히 다른 방향(흑백+포인트 컬러)
2. **타이포그래피 강화** — Pretendard(한) + 영문 폰트 조합으로 전문성 강화
3. **Hero 임팩트** — 비주얼 스토리텔링, 풀스크린, 모션
4. **포트폴리오 큐레이션** — 로고 나열 → 스토리 있는 쇼케이스
5. **기술 스택 전환** — Webflow → Next.js + Tailwind (리뉴얼 범위에 따라 결정)
